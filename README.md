## Instalação

1 - Clonar o repositório:

```
git clone https://github.com/JoaoVictorBezerra/react-portfolio.git
```

2 - Instalar as dependências

```
yarn install
```

3 - Iniciar o projeto

```
yarn dev
```

4 - Buildar a aplicação

```
yarn build
```
